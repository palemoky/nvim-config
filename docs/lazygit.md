# LazyGit

> 以下内容直接在界面按 ? 即可获取

## 全局按键

| 按键               | 功能             |
| ------------------ | ---------------- |
| q                  | 退出             |
| ?                  | 帮助             |
| Esc                | 取消/返回        |
| Tab/左右方向键/1~5 | 切换左侧面板     |
| g                  | reset HEAD       |
| G                  | 打开Pull Request |
| p                  | git pull         |
| P                  | git push         |
|,,/..| 面板中的顶部/底部|


## 文件面板

| 按键  | 功能                  |
| ----- | --------------------- |
| space | 暂存/取消暂存单个文件 |
| a     | 全部暂存/取消暂存     |
| c     | commit                |
| A     | git commit --amend                      |
| d     | Discard（丢弃改动）   |
| Enter | 查看diff              |
| e     | 编辑文件              |


## 分支面板

| 按键  | 功能       |
| ----- | ---------- |
| space | 切换分支   |
| n     | 新建分支   |
| d     | 删除分支   |
| r     | 重命名分支 |
| f     | git fetch  |


## 提交面板

| 按键       | 功能                |
| ---------- | ------------------- |
| space      | 查看差异            |
| r          | Reword 修改提交信息 |
| s          | Squash 压缩提交     |
| f          | Fixup 修复提交      |
| d          | Drop 丢弃提交       |
| e          | Edit 编辑提交       |
| Ctrl + k/j | 交互变基            |
| C          | Cherry-pick         |
