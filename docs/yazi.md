# Yazi

Yazi 是用 Rust 编写的终端文件管理器，基于异步 I/O，速度极快，并自带图片/视频预览。在本配置中通过 `yazi.nvim` 集成进 Neovim。

> 界面内按 `<F1>`（或独立运行时按 `~`）即可查看完整按键帮助

## 在 Neovim 中打开

| 按键        | 功能                                   |
| ----------- | -------------------------------------- |
| `<leader>-` | 在当前文件所在位置打开 yazi            |
| `<leader>cw`| 在 Neovim 的工作目录打开 yazi          |
| `<c-up>`    | 恢复/切换上一次的 yazi 会话            |

> 在 yazi 中选中文件后按 Enter 即可在 Neovim 中打开；`<Esc>`/`q` 关闭。

## 导航

| 按键    | 功能                       |
| ------- | -------------------------- |
| `h`/`l` | 进入上级/下级目录          |
| `j`/`k` | 上下移动                   |
| `g``g`  | 跳到顶部                   |
| `G`     | 跳到底部                   |
| `H`/`L` | 前进/后退到历史目录        |
| `z`     | 用 zoxide 跳转目录         |
| `Z`     | 用 fzf 跳转目录/文件       |

## 选择

| 按键     | 功能               |
| -------- | ------------------ |
| `space`  | 切换选中并下移     |
| `v`      | 进入可视选择模式   |
| `V`      | 取消可视选择       |
| `Ctrl+a` | 全选               |
| `Ctrl+r` | 反选               |

## 文件操作

| 按键    | 功能                         |
| ------- | ---------------------------- |
| `o`     | 打开文件                     |
| `O`     | 交互式选择打开方式           |
| `y`     | 复制（yank）                 |
| `x`     | 剪切                         |
| `p`     | 粘贴                         |
| `P`     | 强制覆盖粘贴                 |
| `d`     | 删除到回收站                 |
| `D`     | 永久删除                     |
| `a`     | 新建文件（以 `/` 结尾建目录）|
| `r`     | 重命名                       |
| `;`     | 执行 shell 命令              |
| `.`     | 显示/隐藏隐藏文件            |

## 查找与过滤

| 按键 | 功能                       |
| ---- | -------------------------- |
| `/`  | 在当前目录查找             |
| `n`/`N` | 下一个/上一个匹配       |
| `f`  | 过滤当前目录               |
| `s`  | 用 fd 按文件名搜索         |
| `S`  | 用 ripgrep 按内容搜索      |

## 标签页与分栏

| 按键      | 功能                   |
| --------- | ---------------------- |
| `t`       | 新建标签页             |
| `1`~`9`   | 切换到第 N 个标签页    |
| `[`/`]`   | 上一个/下一个标签页    |
| `{`/`}`   | 左移/右移标签页        |
| `Ctrl+c`  | 关闭当前标签页         |

## 任务与其它

| 按键 | 功能                     |
| ---- | ------------------------ |
| `w`  | 查看任务管理器（进度）   |
| `~`  | 打开帮助                 |
| `q`  | 退出                     |
| `Q`  | 退出且不写入 cwd-file    |
